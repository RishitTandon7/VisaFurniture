# VisaFurniture: Your Next E-commerce Destination

VisaFurniture is a modern, responsive e-commerce platform designed to offer a seamless shopping experience for furniture. Built with the powerful **MERN stack** (MongoDB, Express.js, React, Node.js), this application provides a robust and scalable foundation for online furniture sales.

## ✨ Key Features

  * **Effortless User Authentication:** Secure and intuitive registration and login for a personalized experience.
  * **Rich Product Catalog:** Showcase your furniture collection with detailed product descriptions and high-quality images.
  * **Streamlined Shopping Cart:** Customers can easily add, update, and remove items, ensuring a smooth purchasing journey.
  * **Intuitive Order Management:** Efficiently track and manage customer orders from placement to fulfillment.
  * **Scalable Architecture:** Built on the MERN stack for performance and easy expansion.

-----

## 🛠️ Technologies Under the Hood

VisaFurniture leverages industry-standard technologies to deliver a high-quality user experience:

  * **Frontend:** React.js, Redux, HTML, CSS, JavaScript
  * **Backend:** Node.js, Express.js
  * **Database:** MongoDB
  * **Authentication:** JWT (JSON Web Tokens)

## 🚀 Getting Started

Setting up VisaFurniture for development is straightforward. Follow these steps to get your local instance running.

### Prerequisites

Ensure you have the following installed on your system:

  * **Node.js & npm** (Node Package Manager)
  * **MongoDB** (Community Server or Atlas)

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/RishitTandon7/VisaFurniture.git
    ```
2.  **Navigate to the Project Directory:**
    ```bash
    cd VisaFurniture
    ```
3.  **Install Backend Dependencies:**
    ```bash
    cd backend
    npm install
    ```
4.  **Install Frontend Dependencies:**
    ```bash
    cd ../frontend
    npm install
    ```

### Configuration

1.  **Create a `.env` file** in the **`backend`** directory.
2.  Add your MongoDB connection URI and a JWT secret key to this file:
    ```
    MONGO_URI=your_mongodb_connection_string_here
    JWT_SECRET=your_strong_random_jwt_secret_here
    ```
    *Replace `your_mongodb_connection_string_here` with your actual MongoDB connection string (e.g., from MongoDB Atlas or your local instance) and `your_strong_random_jwt_secret_here` with a unique, secure string.*

-----

## ▶️ Running the Application

Once configured, you can launch both the backend and frontend:

1.  **Start the Backend Server:**

    ```bash
    cd backend
    npm start
    ```

    The backend API will be accessible at `http://localhost:5000`.

2.  **Start the Frontend Application:**

    ```bash
    cd ../frontend
    npm start
    ```

    The frontend will automatically open in your web browser at `http://localhost:3000`.
